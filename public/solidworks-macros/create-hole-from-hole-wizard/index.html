Objective#

Objective of this article is to learn how to create Holes using ðŸ‘‰ Hole Wizard through SOLIDWORKS VBA Macros in SOLIDWORKS.

We create Holes in 3 steps in general.

  1. Select a ðŸ‘‰ Sketch from we take the Holes location.
  2. Loop through each ðŸ‘‰ Sketch point.
  3. Create Holes using ðŸ‘‰ Hole Wizard feature using HoleWizard method.

This method is not updated method, so use this method if you want to quickly create a new Holes using Hole Wizard.
